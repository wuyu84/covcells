# covcells
 Make cells for figures

函数使用方法
cellsmake(data)

data为输入数据框，其格式要求：   
包括至少7个变量,变量名为:管辖区，街道，采样，报告，姓名，社会面，职业。
最好均为文本格式。   
1.管辖区 变量长度为2，包括深圳11个区划，取2个字，比如罗湖、福田等等。   
2.街道 为深圳市街道，不含‘街道’字样   
3.采样 为采样日期   
4.报告 为阳性报告日期   
5.社会面 包括：社会面、隔离管控 两个分类   
6.职业 为文本格式。   
   
输出会在当前目录下生成一个命名为 "待做图元素库xxxx".pos 的文件，将其导入wps流程图即可。
